# Homebrew Tap for chagg

This is the official [Homebrew](https://brew.sh) tap for [chagg](https://github.com/codested/chagg), a release-note workflow tool for Git repositories.

The formula in this repository is **automatically maintained** by [GoReleaser](https://goreleaser.com) as part of the chagg release process. For documentation, source code, and issue tracking, see the main repository.

**Main repository:** [codested/chagg](https://github.com/codested/chagg)

## Install

```sh
brew install codested/tap/chagg
```

## Update

```sh
brew upgrade chagg
```

## Issues

If you run into installation or upgrade problems specific to Homebrew, please [open an issue](https://github.com/codested/homebrew-tap/issues) here.

For bugs, feature requests, or anything related to chagg itself, use the [main issue tracker](https://github.com/codested/chagg/issues).
